This is a Multi Warehouse Enabled Application, To start, you have to create a WAREHOUSE CODE.
You can have as many WAREHOUSE CODE as you like, each WAREHOUSE CODE will have their own set of data.

Click 'ADD' to add a new WAREHOUSE CODE.

Select a WAREHOUSE CODE and click 'SELECT' to enter the system using the WAREHOUSE CODE's set of data.

You data resides at [%APP_PATH%], please backup them from time to time.
