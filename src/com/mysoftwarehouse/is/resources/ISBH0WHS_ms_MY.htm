Selamat Datang! Penggunaan applikasi ini bermaksud anda telah bersetuju dengan Persetujuan Lesen Pengguna.

Applikasi ini bersifat Berbilang Gudang(Multi Warehouse), untuk mula menggunakan applikasi ini, anda perlulah mencipta satu KOD GUDANG (WAREHOUSE CODE).
Anda boleh mencipta semua KOD GUDANG yang anda suka, setiap KOD GUDANG akan mempunyai set data mereka sendiri.

Tekan 'TAMBAH (ADD)' untuk mencipta KOD GUDANG yang baru.

Pilih satu KOD GUDANG dan tekan 'PILIH (SELECT)' untuk menggunakan set data KOD GUDANG tersebut.

Semua data anda disimpan di [%APP_PATH%], anda dinasihatkan untuk membuat salinan (backup) dari semasa ke semasa.

Anda boleh menukar bahasa di sini.
