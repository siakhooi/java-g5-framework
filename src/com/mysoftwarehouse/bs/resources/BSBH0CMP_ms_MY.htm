Selamat Datang! Penggunaan applikasi ini bermaksud anda telah bersetuju dengan Persetujuan Lesen Pengguna.

Applikasi ini bersifat Berbilang Syarikat(Multi Company), untuk mula menggunakan applikasi ini, anda perlulah mencipta satu KOD SYARIKAT (COMPANY CODE).
Anda boleh mencipta semua KOD SYARIKAT yang anda suka, setiap KOD SYARIKAT akan mempunyai set data mereka sendiri.

Tekan 'TAMBAH (ADD)' untuk mencipta KOD SYARIKAT yang baru.

Pilih satu KOD SYARIKAT dan tekan 'PILIH (SELECT)' untuk menggunakan set data KOD SYARIKAT tersebut.

Semua data anda disimpan di [%APP_PATH%], anda dinasihatkan untuk membuat salinan (backup) dari semasa ke semasa.

Anda boleh menukar bahasa di sini.
